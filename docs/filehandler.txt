File handler docs and testing

Purpose:
Reads in the data files necessary for the program. Should handle all
direct access to files.

Overview:
- Load in files from /data directory
- Upon file request, return file contents as text

Process:
- Get index of file needed to read
- Map index to file name
- Look through /data for matches to file name
- If found, return the contents as string
- If not found, return error

Classes:
FileHandler

Methods:
String readFile(String filename)
List<String> readFileLines(String filename)
List<String> listFiles()

Tests:
1. readFile() with invalid file
    - Returns/throws error

2. readFile() with valid file
    - Returns file contents as String
    - Returns empty string for empty file

3. readFileLines() with invalid filename
    - Returns/throws error

4. readFileLines() with valid filename
    - Returns correct file as list of strings
    - Reads all lines

5. listFiles() when files exist in directory
    - Returns only text files
    - Returns files in sorted order

6. listFiles() when no files in given directory
    - Returns empty list
